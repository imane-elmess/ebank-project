# I&A eBank — Your Money, Your Control

Application bancaire développée dans le cadre du module 
**Architecture des Composants**.

## Technologies
- Backend : Spring Boot, Spring Security, JWT, JPA
- Frontend : React, Vite
- Base de données : PostgreSQL / H2
- Sécurité : JWT, BCrypt
- Email : SMTP
- Rapport : LaTeX

## Fonctionnalités
- Authentification JWT
- Gestion des clients
- Création de comptes bancaires
- Virements et opérations
- Historique des opérations
- Reset / changement de mot de passe
- Sécurité par rôles (Agent / Client)
